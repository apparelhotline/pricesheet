Cowork

How this works. Your Claude Code cowork sessions run on GitHub as claude/* branches with draft PRs. This app mirrors them and lets you reply: posting a message here adds a comment on the session's PR — if that session is watching its PR, it wakes up and responds. It's the real bridge to the official web sessions.
Stored only on this device (localStorage). Use a fine-grained PAT scoped to your repo with Contents: read, Pull requests: read & write, Actions: read — or a classic token with repo scope.
Branches starting with this are treated as cowork sessions. Default claude/.
